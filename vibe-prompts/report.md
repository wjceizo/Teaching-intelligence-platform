# 数学教育智能体平台技术报告（2025 更新版）

## 1. 系统概述

### 1.1 项目背景

随着人工智能技术的快速发展，教育领域正经历深刻的数字化转型。传统的数学教育面临着学生基础差异大、教师答疑负担重、学习反馈不及时等挑战。本项目旨在构建一个融合 AI 技术的智能数学教育平台，通过智能问答、自适应学习、代码实训等功能，提升数学教学质量和学习效率。

### 1.2 项目目标

1. **智能化教学支持**：通过 AI 问答系统提供 7×24 小时的学习辅导
2. **个性化学习路径**：基于认知诊断提供定制化学习建议
3. **协作学习环境**：支持师生互动、笔记共享和问题讨论
4. **多模态学习资源**：整合文本、图表、代码和形式化证明
5. **数据驱动决策**：为教师提供学情分析和教学洞察

### 1.3 设计原则

#### 1.3.1 用户体验原则

- **简洁直观**：采用卡片式布局和清晰的信息层级，降低认知负担
- **响应式设计**：支持桌面（主要）、平板和移动端的自适应布局
- **一致性**：统一的交互模式和视觉语言，减少学习成本
- **可访问性**：符合 WCAG 2.1 标准，支持键盘导航和屏幕阅读器

#### 1.3.2 技术架构原则

- **前后端分离**：前端和后端独立开发、部署和扩展
- **组件化开发**：可复用的 UI 组件库，提升开发效率
- **模块化架构**：功能模块解耦，便于维护和扩展
- **API 优先**：RESTful API + WebSocket 设计，支持多端接入

#### 1.3.3 数据安全原则

- **权限控制**：基于角色的访问控制（RBAC）
- **数据加密**：敏感数据传输和存储加密
- **隐私保护**：遵循 GDPR 和个人信息保护法规
- **审计日志**：关键操作记录可追溯

#### 1.3.4 性能优化原则

- **懒加载**：按需加载组件和资源
- **缓存策略**：合理使用浏览器缓存和 CDN
- **代码分割**：优化首屏加载时间
- **异步处理**：长时间操作采用异步处理和进度反馈

### 2.5 AI 模型集成

#### 2.5.1 RAG 架构

```
用户提问
    │
    ▼
Embedding（text-embedding-3-small）
    │
    ▼
向量检索（Milvus，Top-K COSINE 相似）
    │
    ▼
上下文构建（讲义片段 + 历史对话）
    │
    ▼
Prompt 工程（系统提示 + 上下文 + 问题）
    │
    ▼
LLM 推理（Claude claude-sonnet-4-5 / GPT-4o）
    │
    ▼
答案后处理（Markdown 格式化、LaTeX 渲染、引用来源）
    │
    ▼
返回答案
```

## 3. 系统功能模块

### 3.1 用户管理模块

- **身份认证**：JWT（access token 15 分钟 + refresh token 7 天）、邮箱注册
- **角色权限**：学生、教师、管理员（RBAC）
- **个人中心**：头像、昵称、学习统计、偏好设置

### 3.2 课程管理模块

- **学生端**：课程浏览、章节学习、Markdown + LaTeX 讲义、学习进度跟踪
- **教师端**：课程 / 章节 CRUD、Markdown 编辑器、内容发布
- **管理员**：课程审核、数据统计

### 3.3 问答中心模块

- **AI 答疑**：RAG 检索增强、段落级上下文感知、多轮对话、引用来源
- **教师答疑**：待回答队列、富文本编辑、投票、采纳最佳答案
- **问题管理**：多维筛选、全文搜索、置顶、标记已解决

### 3.4 笔记系统模块

- **一键摘录**：选中讲义文本 → 自动关联章节
- **自由笔记**：Markdown + LaTeX + 代码块
- **组织管理**：按课程分类、标签、全文搜索
- **笔记共享**：私有 / 公开 / 指定用户

### 3.5 代码实训模块

- **Monaco Editor**：语法高亮、自动补全
- **Docker 沙箱**：Python / JS / C++ 安全执行，限制内存和 CPU，禁用网络
- **测试系统**：单元测试结果、执行时间、内存使用
- **提交历史**：代码对比（Diff）、回滚

### 3.6 在线测验模块

- **题型**：单选、多选、填空、简答、证明题
- **组卷**：手动选题 / 自动组卷（按知识点和难度）
- **答题**：倒计时、自动保存、答题卡
- **批阅**：选择题自动评分、主观题手动打分

### 3.7 Lean 4 证明器模块

- **代码编辑器**：Lean 4 语法高亮、Tactics 自动补全
- **实时验证**：类型检查、Goal 显示、Subgoal 状态
- **证明助手**：Tactics 建议、Mathlib 定理库搜索
- **可视化**：证明树、依赖图

### 3.8 学习分析模块

- **学生仪表盘**：学习进度、时长统计、知识图谱、成绩趋势
- **教师视角**：班级概览、学情分析、常见错误、问题热点
- **图表**：折线图、柱状图、雷达图、热力图
